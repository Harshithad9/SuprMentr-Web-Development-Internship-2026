# Assignment (08/04/2026)

## Assignment Name: Role Guard

### Description

This assignment focuses on implementing role-based authorization in a backend application. Users are assigned roles (user or admin), and access to certain routes is restricted accordingly.

---

## Technologies Used

* Node.js
* Express.js
* MongoDB
* JWT
* bcrypt

---

## Features Implemented

* User authentication (signup/login)
* Role-based access control
* Protected routes using middleware
* Admin-only route restriction

---

## Learning Outcome

Through this assignment, I learned how to implement secure authorization systems and control access to backend resources based on user roles.
