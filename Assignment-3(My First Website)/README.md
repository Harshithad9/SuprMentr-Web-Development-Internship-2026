# Assignment (12/02/2026)

## Assignment Name: My First Website

### Description

This assignment focuses on building a simple personal profile webpage using basic HTML elements such as headings, images, links, and lists.

The main objective was to understand how a web page is structured and to practice fundamental HTML concepts by creating a clean and organized layout.

---

## Technologies Used

* HTML5

---

## Features Implemented

* Main heading displaying name and title
* Profile image
* About Me section
* Skills section using an unordered list
* Hobbies section using an ordered list
* External links (GitHub and LinkedIn)
* Footer section

---

## Concepts Practiced

* Basic HTML document structure
* Headings (`<h1>`, `<h2>`, `<h3>`)
* Paragraphs (`<p>`)
* Images (`<img>`)
* Hyperlinks (`<a>`)
* Unordered lists (`<ul>`)
* Ordered lists (`<ol>`)
* Horizontal rule (`<hr>`)

---

## Learning Outcome

Through this assignment, I learned how to create a basic webpage and structure content using HTML. I also understood how to include images and links, and the difference between ordered and unordered lists.
