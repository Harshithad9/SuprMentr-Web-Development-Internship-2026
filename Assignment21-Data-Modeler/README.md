## Assignment Name: Data Modeler

### Description

This assignment focuses on designing a MongoDB schema for a blogging platform. It includes defining models for users, posts, and comments, along with their relationships.

---

## Technologies Used

* Node.js
* MongoDB
* Mongoose

---

## Features Implemented

* Designed User schema
* Designed Post schema with author reference
* Designed Comment schema linked to user and post
* Established relationships between collections
