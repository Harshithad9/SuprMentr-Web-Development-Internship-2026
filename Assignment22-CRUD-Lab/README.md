# Assignment (03/04/2026)

## Assignment Name: CRUD Lab

### Description

This assignment involves implementing CRUD operations using MongoDB and Mongoose. It demonstrates how to create, read, update, and delete data from a database through an API.

---

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose

---

## Features Implemented

* Create new tasks
* Retrieve all tasks
* Retrieve a task by ID
* Update task details
* Delete tasks
* Connected backend with MongoDB

---

## Learning Outcome

Through this assignment, I learned how to integrate MongoDB with a Node.js application using Mongoose and perform CRUD operations on real data.
