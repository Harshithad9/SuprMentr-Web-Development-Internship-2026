# Assignment (06/04/2026)

## Assignment Name: Secure Login

### Description

This assignment focuses on implementing user authentication using hashed passwords and JSON Web Tokens (JWT). It includes signup and login functionality with secure password storage.

---

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* bcryptjs
* jsonwebtoken

---

## Features Implemented

* User signup with password hashing
* Secure login with password verification
* JWT-based authentication
* Protected routes using middleware

---

## Learning Outcome

Through this assignment, I learned how to implement secure authentication systems, protect user data, and manage sessions using JWT.
