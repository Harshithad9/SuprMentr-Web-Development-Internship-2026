# 📘 Assignment (26/03/2026)

## 📚 Assignment Name

Route Master

## 📌 Description

This assignment demonstrates building RESTful routes using Express.js for a bookstore system, managing books and authors.

---

## 🛠️ Technologies Used

* Node.js
* Express.js
* JavaScript

---

## ✨ Features

* Created routes for books and authors
* Implemented GET, POST, DELETE APIs
* Organized routes using Express Router
* Built a simple REST API structure
