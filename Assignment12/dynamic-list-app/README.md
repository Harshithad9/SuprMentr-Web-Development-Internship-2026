# 📘 Assignment (10/03/2026)

## Assignment Name

Dynamic List App

## Description

The Dynamic List App is a React application that allows users to manage a task list. Users can add new tasks and remove tasks dynamically using React hooks.

---

## Technologies Used

* React.js
* JavaScript (ES6)
* HTML5
* CSS3
* Node.js

---

## Features

* Add new tasks to the list
* Delete tasks from the list
* Dynamic UI updates using React state
* Simple and interactive task manager

---

## How to Run

1. Navigate to the project folder

```
cd dynamic-list-app
```

2. Install dependencies

```
npm install
```

3. Start the React development server

```
npm start
```

4. Open the application in your browser

```
http://localhost:3000
```
