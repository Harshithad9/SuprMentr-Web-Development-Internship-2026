# Assignment (07/03/2026)

## Assignment Name
Weather Dashboard

## Description
This React application fetches weather data from the OpenWeather API and displays it in a dashboard format.

## Features
- Search weather by city name
- Fetch data from public API
- Display temperature, humidity, and weather description
- Loading state while fetching data
- Error handling for invalid cities

## Technologies Used
React.js
JavaScript
OpenWeather API
CSS
