## ⚠️ Common Issue Faced

While working with folders, the system may show:

> "Retrieving data. Wait a few seconds and try to cut or copy again."

### 🔍 Reason

* Files are still being processed by the system
* Large folders (like React projects with `node_modules`)
* File indexing or background operations
* Files are in use by another application

---

## ✅ Solutions

* Wait for a few seconds before retrying
* Close applications using the folder (e.g., VS Code, Node server)
* Restart File Explorer
* Avoid moving large folders while processes are running
* Use terminal commands for faster operations
